About Us

Deepfakes are a form of synthetic media that use peep learning algorithms to create fake or manipulated images, audio or video content This media can be very convincing, making it difficult to distinguish between real and fake content. Deepfakes can be created using various techniques such as face swapping, lip syncing, and puppeteering. They have become a growing concern as they can be used to spread false information and impersonate individuals. Our deep learning system is designed specifically to combat deepfakes by using object detection to determine whether an image is real or fake.

Our AI model works by first running a facial detection model using OpenCV. The output of this model is then fed into a convolutional neural network (CNN) which was trained to classify deep faked images. The results are then recorded to the facial detection model which then applies bounding boxed and labels to any faces found within the image. The model will return text on the image indicating if it is “Real” or “Fake.



Ability to detect multiple faces at once

Our model works by using facial recognition on any image submitted and running a deep fake detection model on each face, allowing for multi-face detection.

Use images of any size


No resizing/rescaling required, as long as the face is clearly visible within the image our model will extract and process it.

Accepts most file formats

Our model can accept a wide variety of formats including JPEG, PNG, BMP, WEBP, etc.
